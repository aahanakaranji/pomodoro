# ⏱️ Pomodoro Timer App

A simple and elegant Pomodoro timer to help boost productivity by working in focused intervals with short breaks.

## 🚀 Features

- Pomodoro timer
- Short and long break modes
- Visual countdown

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
